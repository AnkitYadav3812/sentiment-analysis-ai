# AI Sentiment Analyzer

A Natural Language Processing (NLP) web application that analyzes the sentiment of user input text and classifies it as **Positive** or **Negative** using a transformer-based machine learning model.

🚀 Live Demo  
https://sentiment-analysis-ai-awcus8jeokcwxebru6qzaq.streamlit.app/

---

# 📖 Project Overview

Sentiment analysis is a Natural Language Processing technique used to determine the emotional tone behind a piece of text. This project uses a pre-trained transformer model to analyze text sentiment and provide real-time predictions.

The application allows users to input any text and instantly receive sentiment classification along with a confidence score.

---

# ✨ Features

- Real-time sentiment prediction
- Transformer-based NLP model
- Clean web interface using Streamlit

# ⚙️ Installation

Clone the repository


Move to project directory


cd sentiment-analysis-ai


Install dependencies


pip install -r requirements.txt


Run the application


streamlit run app.py


---

# 🖥️ Usage

1. Open the application
2. Enter a sentence in the input field
3. The model will analyze the text sentiment
4. The output will display sentiment and confidence score

Example input:


I love learning artificial intelligence.


Example output:


Positive Sentiment 😊 (Confidence: 0.99)
